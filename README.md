# auth-ui

Browser interface for JorisJonkers.dev authentication and account flows.

## What It Is

`auth-ui` is a Vue/Vite web interface for login, registration, email
confirmation, and TOTP setup flows backed by `auth-api`.

## Local Use

```bash
pnpm install
pnpm test
pnpm dev
```

- API: [auth-api](https://github.com/JorisJonkers-dev/auth-api)
